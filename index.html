Mapic.js client library

This is the open source client library front-end for Mapic.

Please see the wiki for more information.

Install

See mapic/mapic for installation instructions.

Run Test

These commands must be run from within the domain_engine_1 container.

# enter container
docker exec -it localhost_engine_1 fish

# go to mapic.js folder
cd public

# install node modules
npm install

# run tests
npm test
